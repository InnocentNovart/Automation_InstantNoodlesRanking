# Automation_InstantNoodlesRanking
#### About the project:
In this RPA project, I developed a robot that autonomously ranks all countries in the world based on their noodles consumption and noodles preferences. It implements these functions using UiPath tools including data scrapper, Excel tables creation and computation, python scope, documentation, reporting, and email automation.

Data is collected from the open-source portal: https://instantnoodles.org/en/noodles/demand/table/
